KEY FEATURES
═══════════════════════════════════════════════════════════════

✅ Real Google Calendar Integration
   - Sync with your actual Google Calendar
   - Persistent event storage
   - Email invitations to candidates and HR

✅ Smart Interview Scheduling
   - Auto-finds available slots
   - Skips weekends and holidays
   - Avoids scheduling conflicts
   - Timezone intelligent (40+ cities)

✅ Automated Email Reminders
   - Day-of reminders via Gmail API
   - Beautiful HTML email templates
   - Sent to both HR and candidates

✅ Complete CRUD Operations
   - Create, read, update, delete events
   - Search and filter calendar events
   - List multiple calendars

✅ Production Ready
   - OAuth token auto-refresh
   - Error handling and diagnostics
   - Comprehensive documentation


API ENDPOINTS
═══════════════════════════════════════════════════════════════

Calendar:
  GET    /api/calendar/events              # List all events
  GET    /api/calendar/events/{year}/{month} # Events by month
  POST   /api/calendar/events              # Create event
  DELETE /api/calendar/events/{event_id}   # Delete event

Job Description:
  GET    /api/jd-styles                    # Available JD styles
  POST   /api/jd/generate                  # Generate JD

Job Posting:
  POST   /api/posting/post                 # Post job
  GET    /api/posting/status/{job_id}      # Job status
  GET    /api/posting/jobs                 # All jobs

Resume Screening:
  POST   /api/screening/upload             # Upload resume
  POST   /api/screening/screen             # Screen resumes
  POST   /api/screening/shortlist          # Shortlist candidates

Email:
  POST   /api/email/draft                  # Draft emails
  POST   /api/email/send                   # Send emails

Helpdesk:
  POST   /api/helpdesk/query               # Ask question
  GET    /api/helpdesk/load                # Load knowledge base

Onboarding:
  POST   /api/onboarding/send              # Send welcome package

Interview:
  POST   /api/interview/questions          # Generate questions


SETUP CHECKLIST
═══════════════════════════════════════════════════════════════

□ Install Python dependencies
  → pip install -r requirements.txt

□ Setup Google Calendar API
  → Enable API in Google Cloud Console
  → Create OAuth 2.0 credentials (Desktop app)
  → Download credentials.json

□ Configure environment variables
  → Copy .env.example to .env
  → Add GROQ_API_KEY
  → Add OPENAI_API_KEY (for calendar agent)
  → Add HR_EMAIL (your Gmail)

□ Run diagnostics
  → python -m calendar_service.diagnose

□ Start the server
  → python main.py
  → Browser opens for Google login (first time only)

□ Test the integration
  → python test_integration.py
  → Open http://localhost:8000


WHAT'S DIFFERENT FROM MOCK CALENDAR
═══════════════════════════════════════════════════════════════

Before (Mock):                  After (Real Calendar):
─────────────────────────────────────────────────────────────
In-memory storage              → Google Calendar sync
Lost on restart                → Persistent storage
Simple CRUD                    → Smart scheduling
No external integration        → Email invites & reminders
No timezone support            → Multi-timezone aware
Manual conflict detection      → Automatic conflict avoidance
No holiday awareness           → Indian holidays built-in
Fixed time slots               → Dynamic slot finding
